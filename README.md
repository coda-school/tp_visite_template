# TP - carte de visite

## 1 - Démarrage

1. Crée un nouveau projet Github nommé `tp_visite` à l'aide du template https://github.com/coda-school/template-site-statique
2. Clone ce projet sur ta machine et ouvre le dans ton éditeur de code favori
3. Crée un fichier `index.html` et ajoute les éléments d'échafaudage d'une page HTML
4. Ecris le contenu (**sans style CSS pour le moment**) : tu peux utiliser des valeurs factices
  - Nom Prénom
  - Ton activité
  - Numéro de téléphone
  - Adresse e-mail
  - Image d'une photo ou d'un logo
5. Commit et push le code
